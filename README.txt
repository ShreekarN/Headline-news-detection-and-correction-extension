
News Accuracy Checker - Completed Project

How to run:

1) Backend:
   cd backend
   python3 -m venv venv
   source venv/bin/activate    # On Windows: venv\Scripts\activate
   pip install -r requirements.txt
   python app.py

2) Broweser extension:
   - Open extension management
   - Enable Developer mode
   - Load unpacked -> select the 'extension' folder
   - Open any news article page, click the extension icon and 'Check current page'
